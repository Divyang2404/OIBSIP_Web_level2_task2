# 🌟 Tribute Page – Sushant Singh Rajput

## Overview

This project is a tribute webpage dedicated to Sushant Singh Rajput. It highlights his life, education, achievements, career, and inspirational journey using a modern and responsive design.

## Features

- Hero Section
- Biography
- Education & Academic Excellence
- Why I Admire Him
- Career Highlights
- Journey Timeline
- Awards & Achievements
- Image Gallery
- Inspirational Quote
- Legacy Section
- Responsive Design

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Open the project folder.
2. Run `index.html` using Live Server.
